# Next 15 Static SDK Example

This example uses `next@15.5.15` with the OpenNav Next SDK wrapper.

Look at:

- [`next.config.mjs`](./next.config.mjs) for `OpenNavNext(...)`
- [`app/page.js`](./app/page.js) for the static page
- [`app/docs/page.js`](./app/docs/page.js) for a second route
- [`app/guides/setup/page.js`](./app/guides/setup/page.js) for a nested route
- [`package.json`](./package.json) for the pinned dependency versions
