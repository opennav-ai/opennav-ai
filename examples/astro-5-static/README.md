# Astro 5 Static SDK Example

This example uses `astro@5.18.1` with the OpenNav Astro SDK integration.

Look at:

- [`astro.config.mjs`](./astro.config.mjs) for `OpenNavAstro(...)`
- [`src/pages/index.astro`](./src/pages/index.astro) for the static page
- [`src/pages/docs.astro`](./src/pages/docs.astro) for a second route
- [`src/pages/guides/setup.astro`](./src/pages/guides/setup.astro) for a nested route
- [`package.json`](./package.json) for the pinned dependency versions
