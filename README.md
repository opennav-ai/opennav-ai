# OpenNav AI

OpenNav AI is open-source infrastructure for making websites agent-ready.

This repository contains the public packages published under the `@opennav-ai/*` npm scope. The first release target is the static agent-ready site pack: Markdown page artifacts, `llms.txt`, validation, and framework adapters.

## Packages

```txt
packages/
  engine/    Core OpenNav AI generation, indexing, and validation engine
  cli/       Future `opennav` and `opennav-ai` command binaries
```

## Status

Pre-prototype. Interfaces and package boundaries are being established before implementation.
